ContinuumHistogramWriter

ContinuumHistoWriter provides a histogram of solute concentrations at each time point recorded in each instance of the simulation. Data are reported in a tab-delimited text file, with one text file per simulation instance.

MemberTypeDescription
layerStringThe name of the continuum layer on which to report.
occupiedBooleanIf true, only sites that are occupied on the cell layer will be included in the histogram; if false, all sites are included.
