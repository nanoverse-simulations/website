solver

The method by which the continuum state should be updated (solved).

MemberTypeDescription
NonEquilibriumNonEquilibriumSolverA non-equilibrium solver integrates the state of the continuum forward by a specified amount of time.
EquilibriumEquilibriumSolverAn equilibrium solver uses a matrix solver to find the state of the system at equilibrium (i.e., after an infinite amount of time).
