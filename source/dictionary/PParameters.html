Parameters

System-level parameters, such as output directory and number of simulations to run.

MemberTypeDescription
dateBooleanIndicates whether output path should incorporate date and time.
pathStringBase path for simulation output.
seedStringRandom number seed (long integer) or "*" to use current system time (in ms).
instancesIntegerNumber of times to run the simulation.
projectStringThe name of the project (for output path).
maxStepIntegerMaximum number of times to update the simulation state.
