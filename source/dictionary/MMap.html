Map

A map is a 2D visualization consisting of one image per time point, and with visualization coordinates corresponding directly to their simulation coordinate counterparts.

MemberTypeDescription
edgeIntegerSpecifies the size of each lattice site edge (and, by extension, the size of the image as a whole).
outlineIntegerSpecifies the thickness of the outline around each lattice site.
highlightsListSpecifies which highlight channels to visualize, if any, and how they should be visualized.
colorColorModelSpecifies how simulation state should be encoded into color, if at all.
