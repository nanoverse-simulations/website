ContinuumLayer

A continuum layer associates each lattice site with a continuous numerical value. This value can be changed directly, or transformed using a matrix operation.

MemberTypeDescription
boundaryBoundaryThe boundary condition for this layer.
disableOperatorsBooleanIf true, no matrix operations may be scheduled for the simulation. If matrix operations are not needed, disabling them can speed up simulation updates dramatically.
idStringA unique name for the layer.
solverContinuumSolverThe method by which the continuum state should be updated (solved).
