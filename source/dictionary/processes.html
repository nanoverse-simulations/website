processes

A list of top-down nanoverse.runtime.processes to perform at the outset of the simulation.

MemberTypeDescription
DiffuseDiffusionProcessSchedule a homogeneous diffusion process across the entirety of a specified continuum layer.
InjectInjectionProcessSchedule a fixed-value source (injection) at a site or sites of a specified continuum layer.
PowerScatterPowerScatterScatter groups of nanoverse.runtime.cells in a distribution of cluster sizes that may or may not follow a power law. (Check this before using)
DirichletBoundaryEnforcerDirichletBoundaryEnforcerA Dirichlet boundary condition describes region of space within which the value of a field remains constant for all time. This region is often placed along an edge of the system, but it need not be. For example, one might hold the value of the continuum constant in a region at the center of the simulation space. The DirichletBoundaryEnforcer sets the specified coordinates to the desired value, regardless of any other processes that might have otherwise affected them. It must be applied between a Hold process and a Release process.
FillFillFill in a region with agents.
CheckForExtinctionCheckForExtinctionHalt simulation if all sites are vacant.
IntegrateIntegrateUpdate the state of the specified continuum layer.
CheckForFixationCheckForFixationHalt the simulation if only one type of nanoverse.runtime.agent exists.
MockMockProcessLEGACY: Mock process, used in legacy tests.
DivideDivideLEGACY: Divide the nanoverse.runtime.agent to a neighboring site.
CullCullLEGACY: Kill all agents whose health is below a specified threshold.
TriggerTriggerProcessTrigger agents to perform a specified Action.
CheckForDominationCheckForDominationHalt the simulation when the target cell type has the specified fraction of the overall live cell population.
ScatterClustersScatterClustersScatter agents in clusters of a specific size.
TickTickAdvance the simulation clock by the specified dt.
RecordRecordCaptures the state of the simulation. If not specified, occurs by default at the end of each simulation cycle.
OccupiedNeighborSwapOccupiedNeighborSwapSwap the specified nanoverse.runtime.agent with one of its occupied neighbors.
GeneralNeighborSwapGeneralNeighborSwapSwap the specified nanoverse.runtime.agent with any of its neighbors.
ScatterScatterScatter a specified number of new agents to random locations.
CheckForThresholdOccupancyCheckForThresholdOccupancyHalt the simulation of the overall population exceeds a certain threshold.
HoldScheduleHoldBegin queueing changes to the specified continuum layer, but do not execute them until a corresponding "Release" event takes place.
ReleaseScheduleReleaseResolve all changes to the specified continuum layer since the last "Hold" event occurred.
ManualHaltManualHaltImmediately end the current instance of the simulation.
